# Лабораторна робота: Створення презентацій у Beamer (LaTeX)

Цей репозиторій містить матеріали до лабораторної роботи, присвяченої вивченню візуальних стилів та розробці динамічних презентацій за допомогою класу **Beamer**.

---

## 📝 Завдання 1. Дослідження візуальних стилів

Метою завдання є тестування різних тем та колірних схем для вибору оптимального дизайну.

### Алгоритм виконання:
1. **Підготовка:** Відкрити середовище LaTeX (TeXmaker або VS Code).
2. **Клонування:** Завантажити шаблон репозиторію у робочу папку.
3. **Аналіз тем:** Почергово змінювати команду `\newcommand{\mytheme}{...}`, тестуючи теми:
    - `Madrid`, `Berlin`, `Warsaw`, `CambridgeUS`, `Frankfurt`.
4. **Оцінка стилю:** Після кожної компіляції звертати увагу на:
    - Панелі навігації (верхні/нижні).
    - Оформлення списків та блоків.
    - Відображення назв секцій.
5. **Колірні схеми:** Протестувати схеми `\usecolortheme`:
    - `seahorse`, `dolphin`, `beetle`, `crane`.
6. **Висновок:** Обрати найкращу комбінацію та обґрунтувати вибір (наприклад, зручність навігації або контрастність).

```
\section{Висновки}
\begin{frame}{Результати мого дослідження}
    \begin{itemize}
        \item[\checkmark] **Найкраща тема:** [Вписати назву]
        \item[\checkmark] **Найкращий колір:** [Вписати назву]
        \item[\checkmark] **Моє спостереження:** [Наприклад: "Тема Berlin зручна, бо видно всі розділи"]
       
    \end{itemize}
\end{frame}
```
---

## 🚀 Завдання 2. Розробка власної презентації

Створення презентації на 5–7 слайдів з використанням обраної теми.

### Вимоги до презентації:
- [x] **Титульний аркуш:** Назва та дані автора.
- [x] **Зміст:** Автоматичне створення через `\tableofcontents`.
- [x] **Динамічні списки:** Використання ефектів появи елементів (`<1->`, `<2->`).
- [x] **Математика:** Наявність формул.
- [x] **Графіка:** Вставлені зображення або схеми.
- [x] **Блоки:** Виділення тексту через `\begin{block}{...}`.

### Результати:
1. Збережений файл у форматі `.tex`.
2. Скомпільований документ `.pdf`.
3. Завантажені вихідні файли та результат у GitHub-репозиторій.

## 💡 Поради при створенні (Beamer Best Practices)
* **Мінімалізм:** Не перевантажуйте слайди великою кількістю тексту.
* **Списки:** Використовуйте `itemize` замість суцільних абзаців.
* **Баланс:** Гармонійно поєднуйте текст, формули та графічні об'єкти.
* **Логіка:** Забезпечте послідовність викладу матеріалу.
